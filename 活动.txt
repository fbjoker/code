3. 活动发起者，会在活动参与中获得红包奖励，每达到相应奖励积分，即可开启一次红包。最高可获得红包奖励***元

豆蔓微信公众号注册绑定接口文档

    主要接口功能介绍：通过将用户的微信和豆蔓平台账户进行绑定，已有账户的实现微信绑定，没有账户的可通过微信公众号进行注册并绑定，从而实现微信端与豆蔓平台的相互连通，进而实现用户业务上的处理等功能。

微信主菜单：到绑定页面

接口类型：HTTP GET请求
接口地址：http://localhost:8080/CreditMarket/api/v2/users/wechat/toWechatRegisterAndBindingUI

    接口入参：
    名称 	数据类型 	备注 	是否必填
    openid 	String 	微信用户openid 	是
    unionid 	String 	微信用户unionid 	是

    接口出参:
    名称 	数据类型 	备注
    status 	int 	状态(0已绑定，1未绑定，2微信异常)
    openid 	String 	微信用户openid
    unionid 	String 	微信用户unionid

    status说明：
    名称 	数值 	数据类型 	备注
    status 	0 	int 	已绑定
    status 	1 	int 	未绑定，跳转到绑定页面
    status 	2 	int 	微信登录异常

示例：

{"status":0}

{"status":1,
"openid":"oZYDajsDftfShloTkxgnf9cGOwkE",
"unionid":"oXQextzeLwDKNnsV6IKrI8RewmzY"
}

{"status":2}

微信绑定

接口类型：HTTP POST请求
接口地址：http://localhost:8080/CreditMarket/api/v2/users/wechat/wechatBinding

    接口入参：
    名称 	数据类型 	备注 	是否必填
    openid 	String 	微信用户openid 	是
    unionid 	String 	微信用户unionid 	是
    mobile 	String 	用户手机号 	是
    password 	String 	用户密码 	是

    接口出参：
    名称 	数据类型 	备注
    status 	int 	状态(0绑定成功，1用户不存在，2密码错误，3绑定异常)

    示例：
    {"status":0}

微信注册绑定

接口类型：HTTP POST请求
接口地址：http://localhost:8080/CreditMarket/api/v2/users/wechat/doWechatRegisterAndBinding

    接口入参：
    名称 	数据类型 	备注 	是否必填
    mobile 	String 	用户手机号 	是
    password 	String 	用户密码 	是
    captcha 	String 	手机验证码 	是
    openid 	String 	微信用户openid 	是
    unionid 	String 	微信用户unionid 	是

    接口出参：
    名称 	数据类型 	备注
    status 	int 	状态(0注册绑定成功，1手机号无效，2手机已注册，3程序异常，4验证码错误，5注册绑定失败)
    message 	String 	注册绑定的状态信息
    userId 	String 	用户ID

    message说明：
    名称 	数据类型 	备注
    message 	String 	不同的status对应不同的message

    示例：

    {
    "status":0,
    "message":"注册绑定成功!",
    "userId":"0ABD3C66-49C1-41AF-AF5F-FEE9649046F1"
    }

    {"status":1,"message":"手机号无效"}

    {"status":2,"message":"此手机号已注册！"}

    {"status":3,"message":"程序异常!"}

    {"status":4,"message":验证码错误！"}

    {"status":5,"message":"注册绑定失败!"}
